IREWA

IREWA is a full ecosystem energy management platform designed to help users instantly compare heating options and understand real cost savings through data driven insights. Built across web and mobile, the platform combines an interactive cost calculator with professional reporting tools, giving users complete clarity and control over energy decisions. Real time calculations and downloadable reports make energy comparison transparent, efficient, and easy to share.

App Prototype Web Prototype

Problem

Comparing heating options and understanding true energy costs is complex, fragmented, and difficult to validate.

Solution

Built a connected web and mobile ecosystem that delivers real time cost comparisons and generates professional energy reports on demand.

Outcome

  • • Clear visibility into energy savings through real time calculations
  • • Unified experience across web and mobile platforms
  • • Structured, shareable reports that support informed decision making

Key Features

  • Real time heating cost comparison engine
  • One tap PDF report generation
  • Web and mobile ecosystem with shared data layer
  • Secure backend handling calculations and report logic

Tech Stack

Web App: PHP Mobile App: Flutter Backend: Node.js
